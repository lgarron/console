# console.log()

For when you don't want to think about what language you're using, and just get stuff onto the screen.

## Javascript

    console.log("Value:", 2)

## Go

    console.Log("Value:", 2)

## C++

Implementation requires variadic templates (flag: `-std=c++11`).

    console.log("Value:", 2)

## Python

    console.log("Value:", 2)

## Ruby

    console.log("Value:", 2)
